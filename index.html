git学习笔记

#第一章 git认识
git是分布式版本控制系统。多人能在多地同时修改文件，因版本库在个人电脑，所以不需要联网就能修改并保留修改记录，可以随时查看过去的修改记录。多人间通过推送修改的方式到每个人的版本库，通常有一台充当“中央服务器”的电脑，用来方便“交换”大家的修改。
#第二章 git基本命令
###创建版本库
mkdir learngit
cd learngit
git init //通过git init命令把当前目录变成git可以管理的仓库
###添加文件到版本库
注：所有的版本控制系统，智能跟踪文本文件的改动。windows自带的记事本编辑时会在每个文件开头添加0xefbbbf（十六进制）的字符，会遇到很多问题。

第一步，添加文件到仓库

git add read.txt

第二步，提交文件到仓库

git commit -m "说明改动"
